System Permissions

Permissions are the main key to accessing resources on the system. Each role contains a list of these permissions, which effectively grant that role system rights to perform corresponding actions.

  • ReadHome
    Access to view the main dashboard
  • ReadPatients
    Access to view patient information
  • WritePatients
    Access to create and modify patient records
  • ReadUsers
    Access to view system users
  • WriteUsers
    Access to create and modify system users
  • ReadRoles
    Access to view system roles
  • WriteRoles
    Access to create and modify system roles
  • ReadPermissions
    Access to view system permissions
  • ReadAppointments
    Access to view appointments
  • WriteAppointments
    Access to create and modify appointments
  • ReadPharmacy
    Access to view pharmacy information
  • WritePharmacy
    Access to manage pharmacy operations
  • ReadLaboratory
    Access to view laboratory information
  • WriteLaboratory
    Access to manage laboratory operations
  • ReadAuditLogs
    Access to view system audit logs
  • ReadSettings
    Access to view system settings
  • WriteSettings
    Access to modify system settings
